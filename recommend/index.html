基础版相对于免费版有API支持，可以使用基于API的各种插件及客户端等，可使用永久有效的跳转链接，一次上传永久省心。
高级版在使用跳转链接时可以使用我们提供的域名或者自定义域名，免费用户使用的域名为pic.downk.cc，基础版使用的域名为pic.superbed.cn，高级版可自由决定是否使用pic.imgdb.cn或者您自己的域名。您可以在用户中心-自定义域名处进行设置，您也可以将本程序的转发服务部署到您自己的服务器。
专业版相对于基础版主要是一些专有的服务，如更高频次的API请求支持，通过FTP或者替换数据库方式批量上传，离线上传，防盗链，自定义水印等。

小书匠

小书匠应该算是一个不错的Markdown写作工具，我的文章大部分就是在这个网页上操作的（尽管一些语法和Hexo稍有冲突），当然他也有离线版提供下载

小书匠编辑器是一款专为markdown写作而设计的编辑器。

  1. 专为markdown写作设计的文档编辑器，让用户心无旁骛的进行创作。
  2. 多种编辑模式。单栏编辑，双栏编辑，三栏编辑，全屏写作，全屏阅读…想怎么切换，就怎么切换，就是这样随心所欲。
  3. 多种编辑器实现。codemirror编辑器（提供vim,emacs按键，行专注等），ace编辑器（提供vim，emacs按键绑定，显示行号），轻量编辑器，CJK竖排编辑器
  4. 多种主题选择。包括编辑器主题，预览区代码高亮主题，及预览区用户自定义css。
  5. 丰富的语法支持。不仅提供了常用的commanmarkdown语法，还提供了许多有用的扩展语法，比如Latex公式，表格 ，目录， 脚注, 视频, 音频, 附件, checklist, 流程图 等。
  6. 代码块文字格式语法。>
  7. 第三方同步。浏览器存储, 本地文件系统存储, dropbox, evernote, 印象笔记,gitoschina, 为知笔记, github 等多种存储方案，保证了用户数据的安全，也让用户在存储方案上有了更多的选择。
  8. 支持evernote，印象笔记。提供双向操作，可以将文章保存到evernote/印象笔记上，也可以从evernote/印象笔记上导入数据。同时提供标签，附件，图片，待办等相关处理。
  9. 强大的文件管理功能。文件信息，标签，附件，音频，视频，图片管理。
  10. 发布功能。 支持将文章发布到博客平台上。
  11. 邮件发送功能。
  12. sourceMap对照功能。方便在源markdown文件和生成的html文件上进行比较，特别适合markdown初学者使用，了解每一个markdown解析产生的结果，也适用于文章后期的校对上。
  13. ppt。
  14. ppt跨屏演示
  15. pdf预览
  16. typewriter scrolling
  17. autocomplete 和 snippets 功能

音乐安利

걱정말아요 그대（你不要担心）-野菊花

记得当时初三，离中考还剩不到100天，晚上睡觉睡不着的时候，就会打开MP3听听电台，听着听着就睡着了。偶然在FM90.6（好像是），听到了这首歌，当时听着就有直触心灵的感觉，但是一直没找到歌的名字……一直到中考完的之后的几天，闲置在家，和初中同学一起打着游戏，后台放着网易云的每日推荐，听着旋律似曾相识，切出后台……
怎么讲呢，这首歌也很符合当时的意境，现在回想起来真的又像回到了初中那段轻松日子，很怀念吧……就像歌词里：

지나간 것은 지나간대로
过去的事都是过去了

그런 의미가 있죠
这样的意思吧

우리 다함께 노래합시다
我们一起唱歌吧

歌词里的这种意境，也算是对过去的一种释然吧……

网易云的原唱版权限制莫得外链，这首翻唱也不错（但就是只有回忆怀念的样子，并没唱出原唱的那种释然的感觉）。
如果你感兴趣还是推荐去听听原唱걱정말아요 그대（你不要担心）-野菊花(原唱传送门)
